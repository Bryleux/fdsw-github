Acerca de mi

Estoy entusiasmado por la posibilidad de colaborar con un equipo dinámico y creativo para alcanzar metas comunes y crear soluciones impactantes.

HTML

El lenguaje fundamental para la construcción de páginas web, estoy preparado para estructurar y etiquetar contenido de manera efectiva. Mi experiencia incluye el uso de etiquetado semántico para crear sitios web claros y accesibles.

CSS

Lenguaje esencial para dar estilo y diseño a las páginas web, estoy preparado para crear interfaces atractivas y funcionales. Mis habilidades incluyen diseño responsivo y estilización de elementos para proyectos web dinámicos y cautivadores.

Bootstrap

Popular framework de front-end, puedo crear interfaces web modernas y responsivas de manera eficiente. Mi experiencia incluye la utilización de componentes predefinidos y la personalización de estilos para lograr resultados atractivos y funcionales.

Mis proyectos

Iguana Page

Es un sitio web que presenta información detallada sobre las iguanas, destacando sus características físicas y comportamentales. Incluye imágenes ilustrativas y enlaces externos para ampliar el conocimiento sobre estas fascinantes criaturas.

Cuppon Latam

La página web "Cuppon Latam" ofrece una variedad de productos para mascotas con descuentos destacados. Presenta un diseño moderno, menú de navegación intuitivo y un formulario para recibir cupones por correo electrónico.

Delicias Express

"Delicias Express" ofrece una variedad de arepas con sabores tradicionales y únicos. Su sitio web presenta imágenes apetitosas y descripciones detalladas, promoviendo la experiencia culinaria con cada bocado.
